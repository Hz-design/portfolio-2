Internet of Things: Deep Space Tracker

Scope

The conceptualization and experimentation project involves combing IoT with a Deep Space Tracker, which is a motorized mount designed to rotate along with the Earth's rotation. This tracking mechanism allows it to follow deep space objects for extended periods.

Overview:

The experiment involved creating an Internet of Things (IoT) device using Arduino IDE and internet connectivity. Conceptually, it was highly successful, and the added value is evident: while photographing, you can remotely control the camera without physical touch. The system should be fast and extremely accurate, allowing you to swiftly switch to different objects to capture moments efficiently. This enables you to focus less on the technical aspects and enjoy the moment.

However, attempting to create this using Arduino IDE proved to be a completely different experience; connecting various types of APIs to the ESP-32 resulted in a complete failure!
